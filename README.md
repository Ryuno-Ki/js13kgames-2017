# Lost my phone

This is my entry for the js13kgames competition 2017.

## Installation

```sh
git clone https://github.com/Ryuno-Ki/js13kgames-2017.git
cd js13kgames-2017/
npm start
```

## Bundling and shipping

```sh
npm run build
git commit -a
git push
```
